Комплексная система для автоматического матчинга резюме с вакансиями на основе векторного поиска и комбинированного скоринга. Использует модель nomic-embed-text-v1.5 для анализа семантического сходства и встроенное извлечение ключевых навыков.<br>
Основной функционал<br>
Векторный поиск: кодирование текстов резюме и вакансий в многомерное векторное пространство<br>
Комбинированный скоринг: учёт косинусного сходства (60%), перекрытия навыков (25%) и соответствия по длине текста (15%)<br>
Оценка качества: расчёт метрики NDCG@5 на основе экспертных аннотаций<br>
ИИ-ассистент для HR: генерация персонализированных рекомендаций вакансий<br>
Визуализация: 4-панельные графики с анализом распределения метрик<br>
Структура проекта:<br>
├── CV/                         # Папка с резюме в формате .docx
│   ├── 1.docx
│   ├── 2.docx
│   └── ...
├── 5_vacancies.csv                                # CSV с вакансиями
├── ИИ-ассистент для подбора кандидатов            # Основной код
└── evaluation_results.png                         # Выходной график результатов
Основные классы<br>
VacancyResumeMatcher<br>
Основной класс для матчинга резюме и вакансий.<br>
AIAssistant<br>
HR-ассистент для генерации отчётов.<br>
Скоринговая система<br>
Метрика NDCG@5<br>
Нормализованная кумулятивная усиленная выгода учитывает порядок вакансий в ground truth.<br>
Формат входных данных<br>
CSV вакансий (5_vacancies.csv)<br>
Резюме (CV/*.docx)<br>
Стандартные документы Word с текстом резюме в параграфах.<br>
Ground Truth<br>
Функция load_ground_truth() содержит экспертные аннотации в формате:<br>
Ключ: ID резюме (1-30)<br>
Значение: список ID вакансий в порядке релевантности<br>
Визуализация (evaluation_results.png)<br>
4 панели:<br>
График NDCG по резюме — линейная динамика с средним значением<br>
Гистограмма распределения — сколько резюме в каждом диапазоне NDCG<br>
Статистика и качество — основные метрики и итоговая оценка<br>
Кумулятивное распределение — какая доля резюме имеет NDCG ≥ X<br>
